Información General

Propósito del Proyecto

Este sitio web está diseñado para ofrecer una experiencia educativa y visualmente atractiva sobre misiones espaciales. El objetivo es acercar la exploración espacial a estudiantes, curiosos y apasionados del cosmos.

¿Qué encontrarás aquí?

  • Listado de misiones espaciales con imágenes y detalles
  • Información sobre agencias como NASA, ESA y SpaceX
  • Destinos como la Luna, Marte y más allá
  • Fechas clave, logros y curiosidades

Tecnologías utilizadas

Este proyecto está desarrollado con Angular como framework principal, Bootstrap para el diseño responsive, y TypeScript para una estructura robusta. Las imágenes se gestionan mediante rutas locales o base64, y la navegación es completamente SPA.

Beneficios para el usuario

  • Interfaz clara y moderna
  • Acceso rápido a información relevante
  • Navegación fluida sin recargas
  • Contenido visualmente atractivo
